#### English

# MumeiRedstone: Basic PMMP Redstone Features Plugin

## Supported Features

#### Added working Blocks/Tiles
- Doors
	- All type of materials supported
	- Not working with redstone signals
- Lever
- Redstone Block
- Redstone Lamp
- Redstone Repeater
- Redstone Torch
- Redstone Wire
- TNT

#### Added but not working yet
- Button
- Fence Gates
- Pistons
- Pressure Plates
- Trap Doors
